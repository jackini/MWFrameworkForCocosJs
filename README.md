# MWFramework (JAVASCRIPT)

## Introduction

This is a cross-platform game framework for cocos2d-x 3.x jsb.

## Support

- MVC scheme.
- ZIP compression and encryption.
- Sqlite3 database.
- Some common crypto methods.
- JSON format.
- GIF texture.
- Network scheme (HTTP).
